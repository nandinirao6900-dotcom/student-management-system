# 🎓 Student Management System

A beginner-friendly **command-line Student Management System** built with **Python**. This project allows users to add, view, search, and delete student records while demonstrating core Python concepts such as file handling, exception handling, loops, and conditional statements.

---

## ✨ Features

- ➕ Add a new student
- 📋 View all student records
- 🔍 Search for a student by name
- 🗑️ Delete a student record
- 💾 Store student records in a text file (`student.txt`)
- ⚠️ Handle missing files using exception handling

---

## 🛠️ Technologies Used

- Python 3
- File Handling
- Exception Handling
- Loops
- Conditional Statements
- Lists
- String Methods
- `enumerate()`

## 🚀 How to Run

1. Clone the repository:

```bash
git clone https://github.com/nandinirao6900-dotcom/student-management-system.git
```

2. Navigate to the project folder:

```bash
cd student-management-system
```

3. Run the program:

```bash
python student_management_system.py
```
